# ▶️ Getting Started

## 1. Install Python

Check installation:

```bash

```

## 2. Clone the Repository

```bash

```

## 3. Navigate to the Project

```bash
cd python-for-d2c-founders
```

## 4. Run a Python File

```bash
python example.py
```
